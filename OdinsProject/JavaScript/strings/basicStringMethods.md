# Basic String Methods

JavaScript strings are primitive and immutable: All string methods produce a new string without altering the original string.

- **String length**
- **String charAt()**
- **String charCodeAt()**
- **String at()**
- **String [ ]**
- **String slice()**
- **String substring()**
- **String substr()**
